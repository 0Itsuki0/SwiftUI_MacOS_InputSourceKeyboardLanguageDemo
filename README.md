# SwiftUI/MacOS: InputSource / KeyboardLanguage Demo

A demo of working with System-Wide Keyboard Language (Input Sources)
- Get All the Keyboard languages (input sources) that the user has added
- Get properties for the input source (category, id, type, name, selectable, and etc.)
- Get the current input that is selected
- Set the input source programmatically
- Watch for changes.

For more details, please checkout my blog [SwiftUI/MacOS: Get/Set/Watch System-Wide Keyboard Language (Input Sources)]()
